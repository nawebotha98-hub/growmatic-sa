# growmatic-sa
AI Automation Platform for South African Businesses

**The live growmaticsa.com site is no longer built from this repo.** A separate repo, [`nawebotha98-hub/growmaticsa-com`](https://github.com/nawebotha98-hub/growmaticsa-com), is a Lovable-synced React/Vite/shadcn project that already implements the same Claude Design redesign and deploys successfully to GitHub Pages at growmaticsa.com. That repo is the source of truth for the website going forward.

`index.html`/`styles.css`/`script.js`/`assets/` here are a static HTML/CSS/JS recreation of the same design, kept for reference, but no longer deployed — the deploy workflow was removed to stop it generating failed-run emails on every push to this repo.
